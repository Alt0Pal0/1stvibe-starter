# Your 1stvibe project

Built with [1stvibe.ai](https://1stvibe.ai) — a guided weekend with AI teammates that takes you from curious to shipped.

This is the starter template you cloned during Set Up. It's a bare Next.js 15 + Tailwind v4 + TypeScript project — your blank canvas. From here, your 1stvibe coach and Claude Code will help you turn this into your actual idea.

## Quickstart

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) — you should see your welcome page.

## What's next

Head back to your 1stvibe project: <https://1stvibe.ai/app> *(replace this link with your project URL)*

The `/.1stvibe/` directory will fill up automatically as your AI teammates produce artifacts (project brief, requirements backlog, design notes, GTM plan).

## Stack

- [Next.js 15](https://nextjs.org) (App Router)
- [Tailwind CSS v4](https://tailwindcss.com)
- [TypeScript](https://typescriptlang.org) (strict)
- [Biome](https://biomejs.dev) (lint + format)
